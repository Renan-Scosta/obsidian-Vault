# Tags Containers

Ah, as tags de containers HTML! Elas são a espinha dorsal da estrutura de qualquer página web. Pensando nisso, preparei um resumo das principais para você:

**Tags de Seção Semântica (Introduzidas no HTML5):**

- `<header>`: Representa o conteúdo introdutório, geralmente no topo de uma seção ou página. Pode conter logotipos, títulos, navegação, etc.
- `<nav>`: Define uma seção de links de navegação. Destinada a blocos de links importantes para navegar no site.
- `<main>`: Especifica o conteúdo principal e dominante do `<body>` de um documento. Deve haver apenas um elemento `<main>` por página.
- `<article>`: Representa um conteúdo autocontido e independente, como uma postagem de blog, um artigo de jornal ou um widget.
- `<section>`: Agrupa conteúdo tematicamente relacionado. É mais genérico que `<article>`.
- `<aside>`: Representa conteúdo tangencial ao conteúdo principal, como barras laterais, citações ou informações relacionadas.
- `<footer>`: Define o rodapé de uma seção ou página, geralmente contendo informações de direitos autorais, links de contato, etc.

**Tags de Contêiner Genéricas:**

- `<div>`: Um contêiner genérico de divisão ou seção. Não possui significado semântico inerente e é usado principalmente para aplicar estilos (com CSS) ou manipular conteúdo com JavaScript.
- `<span>`: Um contêiner genérico embutido (inline). Similar ao `<div>`, mas usado para agrupar elementos inline para estilização ou manipulação.
